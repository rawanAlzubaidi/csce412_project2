\doxysubsection{Detailed Description}
The \doxylink{struct_request}{Request} struct represents a request from a client to the load balancer. 

This struct defines a request made by a client to the load balancer. It contains the client\textquotesingle{}s input IP address, the output IP address to which the request should be forwarded, and the time duration the request will take to process (in cycles).
